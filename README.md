# Somewhere

Minimal location-based drop discovery app built with Jetpack Compose.

## Requirements
- Android Studio (recommended)
- JDK 17
- Android SDK 26+

## Run (Android Studio)
1. Open the project folder in Android Studio.
2. Sync Gradle.
3. Run the `app` configuration on a device or emulator.

## Build from terminal
If you have Gradle installed locally:

```zsh
gradle :app:assembleDebug
```

> Note: the Gradle wrapper script (`./gradlew`) is not present in this repo.
